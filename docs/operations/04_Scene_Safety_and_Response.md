# Operations — Scene Safety & Response

## Staging / Code 4
- EMS MUST stage until Law Enforcement declares **Code 4**.
- If shots are fired: EMS MUST disengage and re-stage.
- EMS MAY request escort for uncertain scenes.

## Response Modes
Code 1 (Routine) / Code 2 (Urgent) / Code 3 (Emergency) / Code 4 (Secure)

## Evidence
EMS MUST NOT move evidence unless medically required.

## EMS Right to Refuse Unsafe Scene

EMS MUST withdraw if:
- Shots are active
- The scene becomes hostile
- Security is compromised

EMS MAY refuse care if:
- The patient is actively violent
- The environment presents immediate danger

All refusals must be documented.
