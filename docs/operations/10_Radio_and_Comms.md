# Operations — Radio & Comms SOP (Chaos Killer)

## Channels (recommended)
- EMS Primary: dispatch, enroute, on-scene, transport updates
- EMS Command: supervisors, MCIs, admin comms
- Cross-Agency: coordination with LEO/Fire when needed

## Required radio discipline
- MUST keep transmissions short.
- MUST NOT argue on radio.
- MUST acknowledge critical commands (e.g., “stage,” “Code 4,” “MCI”).

## Standard call format
1) Unit ID
2) Location
3) Situation
4) Needs (units, LEO, fire, transport)

Example:
“EMS-12, Route 68, MVC with injuries, requesting Code 4 confirmation and one additional unit.”

## MCI comms
- Command channel for assignments
- Primary channel for patient counts and transport notifications only
