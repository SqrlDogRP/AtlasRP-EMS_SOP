# Injury Recovery Timeline

Minor Trauma:
- Bruises, minor lacerations
- Cleared same day

Moderate Trauma:
- Single GSW, fractures
- 24-48 in-game hours recovery RP

Severe Trauma:
- Multiple GSW, major crash
- Surgery + extended recovery

Catastrophic:
- Massive trauma
- Possible death / long-term injury

AMU sets recovery time.
